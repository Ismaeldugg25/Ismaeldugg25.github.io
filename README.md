## How it's built?
- **HTML, CSS, JavaScript** — frontend structure and styling
- **Python + Jinja** — templating engine that generates the static site from YAML config files
- **GitHub Actions** — automatically builds and deploys the site whenever config files are updated
- **GitHub Pages** — hosts the site for free

## How it works
All personal content lives in the `config/` folder as simple YAML files.
When changes are pushed to the repo, GitHub Actions automatically rebuilds
the site and deploys it to GitHub Pages, no manual deployment needed.
